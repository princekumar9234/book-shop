# Task: Job Portal Web Application

## Objective
Create a full-stack Job Portal Web Application using Node.js, Express, EJS, and MongoDB.

## Requirements
- Two roles: Job Seeker and Employer/Admin.
- Authentication (Register/Login) with bcrypt.
- Session-based authentication.
- Job CRUD for Employers.
- Job Search and Filter for Seekers.
- Job Application with Resume Upload (PDF).
- Dashboards for both roles.
- Responsive and premium UI design.

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JS, EJS.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB with Mongoose.
- **Others**: Multer (file upload), bcryptjs, express-session.
